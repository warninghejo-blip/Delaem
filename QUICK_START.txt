📋 ПЕРВЫЙ ЗАПУСК (один раз):
   1. Запустите SETUP.bat
      → Установит все зависимости автоматически

🔧 ЕЖЕДНЕВНАЯ РАБОТА:

   ✅ ПРОВЕРКА И ИСПРАВЛЕНИЕ КОДА:
      → Запустите DEBUG.bat
      → Автоматически проверит и исправит все проблемы
      → Покажет что нужно исправить вручную (если есть)

   🚀 ДЕПЛОЙ ВСЕГО ПРОЕКТА:
      → Запустите VSE.bat
      → Деплоит Worker И Pages автоматически
      → Показывает URL после успешного деплоя

   ⚡ БЫСТРОЕ ИСПРАВЛЕНИЕ:
      → Запустите AUTO_FIX.bat
      → Автоматически исправит форматирование и простые ошибки

📁 ДОСТУПНЫЕ СКРИПТЫ:

   SETUP.bat      - Первоначальная настройка (один раз)
   DEBUG.bat      - Полная проверка кода и автоисправление
   AUTO_FIX.bat   - Быстрое автоисправление
   VSE.bat        - Деплой Worker + Pages (ВСЁ)
   WORKER.bat     - Деплой только Worker
   INDEX.bat      - Деплой только Pages

💡 СОВЕТЫ:

   • Всегда запускайте DEBUG.bat перед деплоем
   • VSE.bat деплоит ОБА проекта (Worker и Pages)
   • Если что-то не работает - запустите SETUP.bat снова
   • Для локальной разработки: npm run dev
