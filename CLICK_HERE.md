# ✅ НАЙДЕНО! Теперь кликните на функцию в call stack

## Что делать:

### 1. В call stack найдите строку:
```
eh at app-2637cbcbd7da64c9.js:375  ← КЛИКНИТЕ СЮДА!
```

### 2. Или найдите:
```
eF at app-2637cbcbd7da64c9.js:392  ← ИЛИ СЮДА!
```

### 3. Кликните на эту строку

Откроется вкладка **Sources** с кодом функции `eh` или `eF`.

### 4. Найдите в коде строку, где устанавливается x-sign:

Ищите что-то вроде:
```javascript
S.headers["x-sign"] = ei(J);
// или
headers["x-sign"] = ei(J);
// или
this.setRequestHeader("x-sign", ei(J));
```

### 5. Найдите определения функций:

В том же файле найдите:
- `function ei` - функция хеширования
- `function X` или `U.$i` - что добавляется после `@#?.#@`
- `function ep` - функция кодирования
- `new en()` - класс хеша (MD5 или SHA-256?)

### 6. Скопируйте весь код функции и отправьте мне

Скопируйте:
- Функцию `eh` (или `eF`)
- Определения `ei`, `X`, `U.$i`, `ep`, `en`

## Если не можете кликнуть:

1. Откройте DevTools → Sources
2. Найдите файл `app-2637cbcbd7da64c9.js`
3. Перейдите на строку 375 (Ctrl+G, введите 375)
4. Изучите код вокруг этой строки
5. Найдите функцию `eh` или `eF`
