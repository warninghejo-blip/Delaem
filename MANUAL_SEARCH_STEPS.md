# Ручной поиск функций - пошаговая инструкция

## Шаг 1: Откройте файл

1. DevTools → Sources
2. Найдите файл `app-2637cbcbd7da64c9.js`
3. Откройте его

## Шаг 2: Используйте поиск (Ctrl+F)

### Поиск 1: `function ei`
1. Нажмите Ctrl+F
2. Введите: `function ei`
3. Нажмите Enter
4. Если найдено - скопируйте всю функцию (от `function ei` до `}`)

### Поиск 2: `new en()`
1. Нажмите Ctrl+F
2. Введите: `new en()`
3. Нажмите Enter
4. Найдите определение `en` (выше по коду)
5. Скопируйте определение (например: `const en = MD5;`)

### Поиск 3: `function X`
1. Нажмите Ctrl+F
2. Введите: `function X`

## Альтернатива: Поиск по строкам

Если поиск по именам не работает, попробуйте поиск по содержимому:

1. `digest` - найдет функцию хеширования
2. `MD5` или `SHA256` - найдет тип хеша
3. `encodeURIComponent` - найдет функцию ep
4. `x-sign` - найдет где устанавливается заголовок
5. `@#?.#@` - найдет где используется разделитель
